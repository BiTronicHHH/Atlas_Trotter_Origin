## Web application

### Prerequisites

- Yarn (1.22.4 or greater)
- Node.js (v14.16.0 or greater)

### How to start the app

- Create a .env file

#### Development mode

```
# Install all the dependencies
yarn install

# Start the development server
yarn dev
```

#### Production mode

```
# Install all the dependencies
yarn install

# Build the source code
yarn build

# Launch the app
yarn start
```
